Our work on developing guidance for captioning in XR is using the following process:

  1. Identify User Needs
  2. Write Functional Outcomes
  3. Write the Tests
  4. Write Methods to Pass Tests
  5. Write how to and supporting documentation
  6. Write the Guidelines

1. Identify User Needs

We have carried out a task that has taken XR Accessibility User Requirements and identified those that are relevant to captioning. Information on this can be seen on the Captioning XR User Requirements page.
