Atomic Tests

  • ACT-Rules: Image accessible name is descriptive
  • ACT-Rules: Image has non-empty accessible name

Tests

  • This test is measured by the number of img elements in the HTML document or the number of images in non-HTML content being tested.
  • The percentage test result is the number passed divided by the total number of img elements or images.

Procedure [for HTML]

  1. Examine each img element in the content
  2. Check that each img element which conveys meaning contains an alt attribute.
  3. If the image contains text that is not purely decorative, the value of the alt attribute is the same as the text.

Procedure for Technology Agnostic version

  1. Examine each image of text in the content
  2. Check that each image which conveys meaning contains an alt attribute.
  3. If the image contains text that is not purely decorative, the value of the alt attribute is the same as the text.

Expected Results

Check #2 and #3 are true.
