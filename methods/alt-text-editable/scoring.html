Scoring

Critical Failures

  • If the tool allows authors to insert images, and the image is informative or functional, and the author cannot edit the text alternatives.

Rating

  • Make it easy for authors to add text alternatives to their images
  • Make it easy for authors to find the text alternative properties
  • If the text alternative is stored with the image and filled in automatically, the author must still be able to edit the text alternative to adapt it to the context.
Rating Scale
Rating 0 No ability for the author to edit the text alternative
Rating 1 (WYSIWYG) The text alternative can be edited in the source code
Rating 2 (WYSIWYG) The text alternative can be edited in a properties window, but the window is in a deeper level (or Advanced section) than basic image properties.
Rating 3 (WYSIWYG) The text alternative can be edited in a properties window, but the window is in a deeper level (or Advanced section) than basic image properties AND there is a direct link to the properties dialog from a Help function.
Rating 4
  • (Source code editor) The text alternative can be edited in the source code
  • (WYSIWYG) The text alternative can be edited in a properties dialog AND the dialog is at least as prominent as the basic image properties.
