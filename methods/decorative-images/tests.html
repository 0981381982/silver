Atomic Tests

  • ACT Rules 23a2a8: Image has non-empty accessible name
  • ACT Rules e88epe: Image not in the accessibility tree is decorative

Procedure to test if an image is a CSS background image

Test Procedure

  1. Check for the presence of decorative images.
  2. Check that they are included as CSS background images.

Expected Results

If #1 is true then #2 is true.

Procedure to test if an image is CSS generated content

Test Procedure

  1. Check for the presence of decorative images.
  2. Check if decorative images are included using CSS generated content.
  3. Check that the image has an empty alt value.

Expected Results

If 1 is true then 2 and 3 are true.

Procedure to test if an image uses null alt text and no title attribute for decorative images

Test Procedure

  1. Check for the presence of decorative images
  2. Check that the title attribute is either absent or has no value.
  3. Check that the alt attribute is present and has no value.

Expected Results

If 1 is true then 2 and 3 are true.
